Above is a navigation bar that gives the user easy access to the different pages of the website. Personalizing the bar is very simple: replace the # after href with the url of the desired webpage and rename the value of the link.

A key benefit to using Foundation is that everything is responsive. When users hover over any picture or element, there is a response, making Foundation very user-friendly. It also is responsive to mobile and various desktop sizes. From an 11-inch MacBook to a 50-inch 4K 120Hz monitor, Foundation will change accordingly.


Each of these pictures to the right can be easily personalized by replacing the src code with the urls of the desired images. They can also be made into links by changing the href url to the index of the image in your directory.

Another special feature of Foundation is its XY grid system. This gives the user easy custimization of columns and rows, facilitating the creation of tables and other structures. Each cell in the column or row can be easily adjusted to the user's preference with characteristics including background color, font color, size, height, and more.

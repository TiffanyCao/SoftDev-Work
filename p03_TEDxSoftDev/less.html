A CSS Preprocessor

By Tiffany Cao and Ethan Chen

What is LESS?

Less is a style sheet language that implements the use of coding to extend the capabilities of CSS. Essentially, it is like a cross between Python and CSS, so that it is easier to maintain and customize your style sheets.

How to Use

A CSS preprocessor has two parts:
  1. the language
  2. the compiler

The language is the LESS code itself, while the compiler is what turns LESS code into standard CSS. Compilers include Ruby Gem, CodeKit (an OS X app), less.js, and more.

Using less.js is convenient because it essentially works the same way as a CSS stylesheet.

<link rel = "stylesheet/less" href = "/stylesheets/main.less" type = "text/css"/>
<script src = //cdnjs.cloudflare.com/ajax/libs/less.js/3.9.0/less.min.js"></script>
